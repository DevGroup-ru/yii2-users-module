yii2-users-module
=================

Users and RBAC module for Yii2

## Base Routes

### RBAC
- list: `/users/rbac`
- edit: `/users/rbac/update?params`
- delete: `/users/rbac/delete?params`

### Users
- list: `/users/manage`
- edit: `/users/manage/edit?params`
- delete: `/users/manage/delete?params`
